# Antergos NeXT

**Not just a revival — a modernization.**

<br>

<span style="color: #89b4fa;">⬤</span> <span style="color: #a6e3a1;">⬤</span> <span style="color: #f9e2af;">⬤</span>

<br>

</div>

This is an independent continuation of the Antergos Linux distribution, picking up where it left off and pushing forward.

Some things return. Some things don't. We fix what broke, adopt what's new, and leave what's past in the past. This is not a museum piece — Antergos NeXT is a living project that evolves.

<br>

> <span style="font-style: italic; color: #89b4fa;">"I think it's fine so long as it's clear that your project is your own and not affiliated with the original. Have fun!"</span><br>
> — Dustin (original Antergos founder)

<br>

---

<br>

| What's back | What's new | What's not coming back |
|---|---|---|
| The **Antergos name**, logo, Cnchi installer | **GTK4 port** of Cnchi | 1:1 replica of the old ISO |
| Arch live env that just works | **Calamares** experimental installer | Unmaintainable packages |
| Community-driven | **Channel 666** easter egg | Outdated design choices |
| | CI-built package repo | |
