# Antergos NeXT

**Not just a revival — a modernization.**

This is an independent continuation of the Antergos Linux distribution, picking up where it left off and pushing forward.

Some things return. Some things don't. We fix what broke, adopt what's new, and leave what's past in the past. This is not a museum piece — Antergos NeXT is a living project that evolves.

This project is **not affiliated with, endorsed by, or connected to** the original Antergos project or its developers. It is an independent effort built upon the legacy of Antergos, not a sanctioned continuation.

> *"I think it's fine so long as it's clear that your project is your own and not affiliated with the original. Have fun!"* — Dustin (original Antergos founder)

## What's back
- The **Antergos name**, the **logo**, the **Cnchi installer**
- The spirit of an Arch Linux live environment that just works
- Community-driven development

## What's new or changed
- **GTK4 port** of Cnchi (modern toolkit, not the old GTK3)
- **Calamares** as an experimental secondary installer
- **Antergos NeXT memes** package (channel 666 easter egg — type 666 three times during install)
- Modern Arch packaging with CI-built package repository
- Light blue theme, updated wallpapers
- GNOME as the default desktop environment

## What's not coming back
Some original Antergos features, packages, or design choices will not return — either because they no longer serve a purpose, are unmaintainable, or simply belong to a different era. If you came looking for a 1:1 replica, you may be disappointed. That's not the goal here.

## Repos
- **Antergos-NeXT-ISO** — Live ISO build (Arch Linux, GNOME, Cnchi installer)
- **Cnchi** — GTK4 graphical installer
- **antergos-pkgs** — Custom package repository

## Fun fact
This project was originally launched as "Antergos NeXT", temporarily renamed to "Pulsar Linux" after a takedown request, and then restored to "Antergos NeXT" when Dustin Falgout (one of three original devs) gave his blessing. Pulsar Linux continues as a [separate project](https://github.com/Pulsar-Linux).
