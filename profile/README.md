# Antergos NeXT

A community revival of the Antergos Linux distribution.

This project is **not affiliated with, endorsed by, or connected to** the original Antergos project or its developers. It is an independent continuation built upon the legacy of Antergos.

> "I think it's fine so long as it's clear that your project is your own and not affiliated with the original. Have fun!" — Dustin (original Antergos founder)

## Repos
- **Antergos-NeXT-ISO** — Live ISO build (Arch Linux, GNOME, Cnchi installer)
- **Cnchi** — GTK4 graphical installer
- **antergos-pkgs** — Custom package repository

## Fun fact
This project was originally "Antergos NeXT" before being temporarily renamed to "Pulsar Linux". After Dustin gave his blessing to revive the Antergos name, we returned to it. Pulsar Linux continues as a [separate project](https://github.com/Pulsar-Linux).
